# Estudo-Bradesco-Python
📚 Estudos de Python - Curso Fundação Bradesco (18h)

Este repositório reúne os projetos práticos e a base de conhecimento construída ao longo de 18 horas de imersão total no curso de **Python** da Fundação Bradesco. 

O foco principal deste treinamento foi o desenvolvimento do raciocínio lógico e a transição da teoria para a prática, transformando conceitos abstratos de programação em soluções estruturadas dentro do ambiente de desenvolvimento.

---

## 📝 Resumo da Jornada

Durante este curso, a jornada foi traçada desde os primeiros passos com a sintaxe da linguagem até a autonomia para criar programas que interagem diretamente com o usuário e manipulam dados externos. 

Mais do que apenas decorar comandos, o aprendizado foi focado em entender **como pensar como uma pessoa desenvolvedora**: estruturar fluxos de decisão inteligentes, automatizar tarefas repetitivas para otimizar processos e garantir a persistência de informações salvando e organizando dados diretamente no sistema de arquivos do computador.

Este espaço funciona como o meu portfólio inicial em Python, refletindo horas de dedicação, testes, resolução de problemas e a consolidação de uma base sólida para a criação de softwares eficientes.
